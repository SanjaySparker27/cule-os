Pixhawk 6X Pinout

Pixhawk 6X is the latest FMUv6X standard flight controller.

Note: Documentation for Pixhawk 6X is being expanded. For now, refer to the official Pixhawk 6X documentation.

Key Differences from ATHENA H743

  • Dual IMU redundancy (ICM-42688-P + ICM-20649)
  • Triple barometer redundancy
  • FMUv6X standard pinout
  • Ethernet support on some variants

Pixhawk 6C Pinout

Pixhawk 6C is the cost-optimized FMUv6C standard flight controller.

Note: Documentation for Pixhawk 6C is being expanded. For now, refer to the official Pixhawk 6C documentation.

Key Features

  • Single IMU (ICM-42688-P)
  • FMUv6C standard pinout
  • Cost-optimized design
  • Compatible with most Pixhawk accessories
