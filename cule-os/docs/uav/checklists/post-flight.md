# Post-Flight Checklist

## Immediate Post-Landing

```
□ Disarm immediately
□ Check for damage
□ Record landing time
□ Secure aircraft
```

## Data Management

```
□ Download flight logs
□ Export QGC logs
□ Backup data
```

## Battery Care

```
□ Check voltage
□ Allow to cool
□ Storage charge if needed
```

## Documentation

```
□ Log flight details
□ Note any issues
□ Plan maintenance
```
