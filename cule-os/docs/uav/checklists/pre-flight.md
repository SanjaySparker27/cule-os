# Pre-Flight Checklist

## Pre-Flight Preparation

### Weather Assessment

```
□ Wind speed < vehicle rating
□ Visibility > 3 km
□ No precipitation
□ Temperature within battery limits
□ Check NOTAMs and airspace
```

## Physical Inspection

```
□ Visual inspection complete
□ Props balanced and secure
□ Motors spin freely
□ Battery secured
```

## Electronics Check

```
□ Flight controller LED status OK
□ GPS 3D fix acquired
□ RC transmitter bound
□ Telemetry link established
```

## Final Pre-Flight

```
□ Area clear of people
□ Emergency procedures reviewed
□ Timer started
□ GCS recording confirmed
```

## Takeoff

```
□ Arm vehicle
□ Verify control response
□ Increase throttle smoothly
□ Check for drift at 3m
```
