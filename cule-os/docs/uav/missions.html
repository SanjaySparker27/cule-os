Mission Planning

Create, upload, and execute autonomous flight missions with Cule OS

Overview#

Cule OS uses industry-standard MAVLink mission format compatible with QGroundControl and Mission Planner. Missions consist of waypoints with associated commands and parameters.

Prerequisites: Before planning missions, ensure GPS lock and set appropriate geofence parameters.

Mission Types#

Type Description Best For
Survey Parallel flight lines with camera triggering Mapping, photogrammetry
Waypoint Custom path through specific locations Inspection, patrol
Circuit Loiter pattern around a point Surveillance, training
Delivery Point-to-point with payload actions Package delivery

Creating Missions#

Method 1: Command Line#

Create missions using the Cule CLI tools:

# Create a simple waypoint mission
cule-mission create \
  --name "Building Inspection" \
  --altitude 50 \
  --speed 5

# Add waypoints
cule-mission add-waypoint 37.7749 -122.4194
cule-mission add-waypoint 37.7750 -122.4195
cule-mission add-waypoint 37.7748 -122.4193

# Save mission
cule-mission save inspection.plan

Method 2: QGroundControl#

  1. Connect to your drone via telemetry radio
  2. Open the Plan tab in QGroundControl
  3. Click on the map to add waypoints
  4. Configure waypoint actions (camera trigger, delay, etc.)
  5. Upload mission to vehicle

  "version": 1.0,

Executing Missions#

⚠️ Safety Check: Always verify mission altitude is clear of obstacles. Set RTL altitude higher than any obstacles in the mission area.

Upload Mission#

# Upload mission file
cule-mission upload inspection.plan

# Verify mission loaded
cule-mission verify

Execute Mission#

# ARM and start mission
cule-arm --mode auto

# Monitor progress
cule-status mission

# Watch telemetry
cule-telemetry --follow

Mission Control#

During mission execution, you can pause, resume, or abort:

# Pause at current waypoint
cule-mission pause

# Resume mission
cule-mission resume

# Skip to next waypoint
cule-mission skip

# Change speed
cule-mission speed 10

# Abort and RTL
cule-mode rtl

Survey Missions#

For mapping and photogrammetry:

# Create survey grid
cule-mission survey \
  --area "37.7749,-122.4194,37.7759,-122.4184" \
  --altitude 80 \
  --overlap 80 \
  --sidelap 70 \
  --speed 12 \
  --camera-trigger \
  --save survey.plan
Tip: Use cule-mission simulate to preview your mission before flying.
