# ESC Calibration Guide

## Overview

ESC calibration ensures all motors respond identically to throttle commands.

## Calibration Methods

### Method 1: QGroundControl

```
Procedure:
1. Remove propellers
2. Connect battery
3. Go to Power tab
4. Click "ESC Calibration"
5. Set throttle to max, wait for beeps
6. Set throttle to min, wait for beeps
```

## Troubleshooting

### Motor Not Spinning

```
Check:
1. ESC signal wire connected
2. Safety switch pressed
3. Vehicle armed
```
