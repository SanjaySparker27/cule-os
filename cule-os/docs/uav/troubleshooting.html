Troubleshooting

Common issues and solutions for Cule OS UAV Edition

Boot Issues#

Drone won't boot ERR_BOOT_001#

Symptoms: No lights, no beeps, no response when powered

Solution

  1. Check battery voltage (should be 11.1V+ for 3S, 14.8V+ for 4S)
  2. Verify power module LED is on
  3. Check main power connector is seated firmly
  4. Try a different battery
# Check power status
cule-status power
# Expected: Voltage > 11V, Current < 1A

Boot loop (continuously restarting) ERR_BOOT_002#

Symptoms: Beeps repeatedly, LED cycles through colors

Solution

  1. Check SD card is properly inserted
  2. Reflash SD card with latest Cule OS image
  3. Verify SD card is Class 10 or faster
  4. Try a different SD card

GPS Issues#

No GPS fix ERR_GPS_001#

Symptoms: GPS LED not solid, "No GPS" in status

Solution

  1. Move to open area with clear sky view
  2. Wait 2-5 minutes for cold start
  3. Check GPS module connection
  4. Verify GPS is enabled in config
# Check GPS status
cule-status gps

# Expected output:
# Satellites: 8+
# HDOP: < 2.0
# Fix Type: 3D

Poor GPS accuracy (high HDOP) ERR_GPS_002#

Symptoms: HDOP > 2.0, position jumps around

Solution

  1. Move away from tall buildings or trees
  2. Check GPS module mounting (clear view of sky)
  3. Verify GPS module is not near RF interference sources
  4. Consider using GPS with external antenna

Flight Issues#

Cannot arm motors ERR_ARM_001#

Symptoms: Beeping when trying to arm, motors don't spin

Solution

Check the arming checks:

# Check arming status
cule-status arm

# Common issues:
# - GPS: Need 3D fix
# - Compass: Need calibration
# - Accel: Need calibration
# - RC: Need radio calibration

Toilet bowling (circular drift) ERR_FLT_001#

Symptoms: Drone circles while trying to hold position

Solution

  1. Recalibrate compass
  2. Check for magnetic interference near compass
  3. Verify GPS and compass orientations match
  4. Reduce LOITER PID values if needed
# Recalibrate compass
sudo cule-calibrate compass

Excessive vibration ERR_FLT_002#

Symptoms: Jello effect in video, unstable flight, high vibration errors

Solution

  1. Check props for damage or imbalance
  2. Tighten all screws on motors and frame
  3. Balance propellers
  4. Add vibration dampening to flight controller
  5. Check motor mount screws

Telemetry Issues#

No telemetry connection ERR_TLM_001#

Symptoms: No data in ground station, "Disconnected" status

Solution

  1. Verify telemetry radio is powered (LED should be on)
  2. Check both radios are on same frequency
  3. Verify correct COM port is selected in ground station
  4. Try re-pairing radios
# Check telemetry status
cule-status telemetry

# Test radio link
cule-telemetry test

Log Analysis#

Download and analyze flight logs for detailed diagnostics:

# Download latest flight log
cule-logs download --latest

# List all logs
cule-logs list

# Analyze specific log
cule-logs analyze /var/log/cule/flight_001.bin
