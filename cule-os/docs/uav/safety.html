Emergency Procedures#

Lost Control / Flyaway#

  1. Switch to RTL mode immediately
  2. If RTL fails, switch to LAND mode
  3. If still no response, use emergency kill switch (if equipped)
  4. Clear the area and warn others
# Emergency RTL
cule-mode rtl

# Emergency Land
cule-mode land

# Emergency Disarm (cut motors)
cule-disarm --emergency

Low Battery Emergency#

  1. Land immediately - do not attempt RTL if battery is critical
  2. Find nearest safe landing spot
  3. Use LAND mode for controlled descent
  4. After landing, disconnect battery
Battery Levels: Warning at 30%, Critical at 20%, Emergency land at 15%

GPS Loss in Flight#

  1. Switch to ALT_HOLD or STABILIZE mode
  2. Fly manually back to takeoff point
  3. Do not use LOITER, AUTO, or RTL modes without GPS
  4. Land as soon as safely possible

Failsafe Configuration#

Configure automatic responses to system failures:

# /etc/cule/failsafe.conf

# Radio Failsafe (RC loss)
FS_THR_ENABLE=1          # Enable radio failsafe
FS_THR_VALUE=975         # PWM threshold for loss detection

# Battery Failsafe
FS_BATT_ENABLE=1         # Enable battery failsafe
FS_BATT_VOLTAGE=10.5     # Critical voltage (10.5V for 3S)
FS_BATT_MAH=500          # Reserve capacity in mAh

# GCS Failsafe (ground station loss)
FS_GCS_ENABLE=1          # Enable GCS failsafe
FS_GCS_TIMEOUT=5         # Timeout in seconds

# Action on failsafe (0=Disabled, 1=RTL, 2=Land, 3=SmartRTL)
FS_ACTION=1

Geofence Configuration#

Set boundaries to prevent flyaways:

# Enable cylindrical geofence
FENCE_ENABLE=1
FENCE_TYPE=3             # Altitude + Circle

# Altitude limit (meters)
FENCE_ALT_MAX=120

# Circular boundary (meters from home)
FENCE_RADIUS=500

# Action when fence breached (0=Report, 1=RTL, 2=Land)
FENCE_ACTION=1
