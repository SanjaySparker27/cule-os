Installation Steps

1

Download the Image

Download the appropriate image for your hardware platform above.

2

Flash to SD Card

Use BalenaEtcher (recommended) or command line:

xzcat cule-os-axon-jetson-v1.0.img.xz | sudo dd of=/dev/sdX bs=4M status=progress
3

First Boot

Insert SD card, connect flight controller via USB, and power on. Wait 2-3 minutes for first boot.

4

Configure

Connect to WiFi network "Cule-OS-Setup" and SSH:

ssh cule@192.168.4.1
# Password: culeos
5

Run Setup Wizard

sudo cule-config

Follow the prompts to configure your vehicle type and sensors.
